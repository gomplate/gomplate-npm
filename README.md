# gomplate-npm

Installs [gomplate](https://github.com/hairyhenderson/gomplate) with `npm` using [go-npm](https://github.com/sanathkr/go-npm).

## Publishing

Update `version` field in `package.json` with the latest release of `gomplate`. Then run:

```
npm publish
```
